## Compatible with:
    - Magento 2

## Install TheMarketer
```shell
cd path_to_the_store_root_folder
sh mktr/mktr.sh install
```

## unInstall TheMarketer
```shell
cd path_to_the_store_root_folder
sh mktr/mktr.sh uninstall
```
